# HPF.kz · Brand Spec
> 采集日期：2026-04-24
> 资产来源：当前代码库、公开路线资料、项目现有 HPF 标识
> 资产完整度：部分。外部搜索未找到稳定可验证的 HPF.kz 官方品牌资产，当前以项目内资产作为临时品牌根基。

## 核心资产

### Logo
- 主版本：`public/brand/hpf-mark.svg`，透明底纯字标，仅保留 `HPF KZ`
- Favicon：`public/favicon.svg`
- 使用场景：导航、页脚、OG 图、深浅底识别
- 禁用变形：不要拉伸，不要加描边，不要改成渐变色

### 主视觉资产
- 路线图：代码生成的路线 atlas，用于表达中国、阿拉木图、库雷克、巴库、欧洲的运营链路
- 物流照片：当前使用 Unsplash 物流/仓储/铁路图作为临时素材
- 后续建议：替换为 HPF 自有仓库、车队、库雷克港、阿拉木图运营现场照片

## 签名细节
- 第一屏必须出现路线 atlas，而不只是装饰图
- 信息卡片像运营单据，不像营销卡片
- 用节点、坐标、状态、风险层级表达“确定性”

## 气质关键词
- 中亚通道
- 运营确定性
