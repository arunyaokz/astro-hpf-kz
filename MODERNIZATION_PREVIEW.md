### 核心价值卖点

#### 之前
1. **可解释的时效**: 承诺以SLA为锚，路径、缓冲与口岸波动参数透明呈现
2. **系统化合规**: 合规不是额外工作，而是流程默认；一次性设计优先于事后补救
3. **生态协同**: 以接口为纽带联接港口、机场、铁路、车队、保险与金融伙伴

#### 现在 ✨
1. **实时可视化**: 全程追踪每一个货物，透明化每一个环节，让物流状态尽在掌握
2. **极速通关**: 库雷克港绿色通道，专业清关团队，平均节省40%时效
3. **智能调度**: AI驱动的路线优化，多式联运智能切换，确保最优成本和时效

### 核心服务

#### 之前
- **库雷克绿通**: 跨里海关键节点的优先通行方案
- **清关合规**: 哈萨克斯坦&EAEU技术法规对接
- **仓储分拨**: 保税/普通/温控，SKU级可视化
- **干线联运**: 海/铁/公/空多式联运协同
- **最后一公里**: B2B配送与电商逆向闭环
- **金融增值**: 保险、结算、资金流优化

#### 现在 ✨
- **库雷克绿色通道**: 跨里海核心港口优先通行，缩短40%中转时间
- **一站式清关**: 哈萨克斯坦&EAEU全境专业清关服务
- **智能仓储**: 多温区仓储网络，实时库存可视化管理
- **多式联运**: 海运/铁路/公路/空运智能组合，最优成本时效
- **终端配送**: 覆盖中亚全境的最后一公里配送网络
- **供应链金融**: 灵活的支付方案、货运保险、资金优化服务

### 公司定位

#### 之前
- **使命**: 用系统化确定性，为客户创造长期价值
- **愿景**: 成为跨里海通道最可信赖的端到端物流平台
- **价值观**: 坦诚透明｜长期主义｜合规先行｜以客户成功为北极星

```

     #8B5CF6 (Purple)

```

#### 渐变系统
```css
/* 新增 5 种流体渐变 */
--gradient-primary: linear-gradient(135deg, #6366F1 0%, #8B5CF6 50%, #EC4899 100%);
--gradient-secondary: linear-gradient(135deg, #06B6D4 0%, #3B82F6 50%, #6366F1 100%);
--gradient-accent: linear-gradient(135deg, #10B981 0%, #06B6D4 100%);
--gradient-warm: linear-gradient(135deg, #F59E0B 0%, #EF4444 50%, #EC4899 100%);
--gradient-cool: linear-gradient(135deg, #3B82F6 0%, #06B6D4 50%, #10B981 100%);

/* Mesh 渐变背景 */
--gradient-mesh-1: 多层径向渐变组合
--gradient-mesh-2: 多层径向渐变组合
```

### 字体系统

#### 之前
- 西文: SF Pro Display / SF Pro Text
- 中文: PingFang SC / Noto Sans SC
- 最大字号: 4.5rem (72px)

#### 现在 ✨
- 西文: **Inter** (更现代、专业)
- 中文: Noto Sans SC
- 最大字号: **8rem (128px)**
- 流体字体: 完美响应式缩放

### 新增设计元素

#### 1. 磨砂玻璃效果 (Glassmorphism)
```css
.glass-card {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  box-shadow: 0 20px 25px rgba(0, 0, 0, 0.1);
}
```

#### 2. 3D 深度效果
```css
.card-3d:hover {
  transform: rotateX(5deg) rotateY(5deg) translateY(-10px);
}

/* 多层阴影 */
--shadow-3d:
  0 1px 1px rgba(0,0,0,0.12),
  0 2px 2px rgba(0,0,0,0.12),
  0 4px 4px rgba(0,0,0,0.12),
  0 8px 8px rgba(0,0,0,0.12),
  0 16px 16px rgba(0,0,0,0.12);
```

#### 3. 彩色阴影
```css
/* 品牌色阴影 */
--shadow-brand: 0 10px 40px -10px rgba(99, 102, 241, 0.4);
--shadow-accent: 0 10px 40px -10px rgba(6, 182, 212, 0.4);
```

#### 4. 按钮动画
```css
/* 光泽扫过效果 */
.btn::before {
  /* 渐变光泽从左向右扫过 */
  transform: translateX(-100%);
}

.btn:hover::before {
  transform: translateX(100%);
}
```

### 组件对比

| 组件 | 之前 | 现在 |
|------|------|------|
| **按钮** | 单色背景 | 渐变背景 + 彩色阴影 + 光泽动画 |
| **卡片** | 简单阴影 | 玻璃态 / 3D变换 / 悬停抬升 |
| **标题** | 常规大小 | 超大字号 + 负字间距 + 渐变文字 |
| **背景** | 纯色 | Mesh渐变 / 流体渐变 |
| **交互** | 基础过渡 | 弹性动画 + 微交互 |
